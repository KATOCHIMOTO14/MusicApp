# 🎵 MusicApp — Музыкальное приложение

MusicApp — это десктопное приложение для Windows, разработанное на платформе .NET (C#) с использованием Windows Forms. Оно позволяет воспроизводить MP3-файлы, управлять музыкальной коллекцией, создавать плейлисты и получать персональные рекомендации на основе предпочтений пользователя.

## 🚀 Основные возможности

### Для всех пользователей
- Регистрация с выбором любимых жанров
- Вход в систему с хешированием паролей (BCrypt)
- Просмотр списка треков в таблице (название, исполнитель, альбом, длительность)
- Фильтрация треков по исполнителю и альбому
- Персональные рекомендации на основе любимых жанров
- Воспроизведение MP3 с полным управлением:
  - Play / Pause / Stop
  - Next / Prev (переключение треков)
  - Shuffle (перемешивание)
  - Регулировка громкости
  - Перемотка (ползунок позиции)
- Автоматическое переключение на следующий трек
- Сохранение и восстановление позиции прослушивания
- Отображение обложек альбомов
- Создание и управление плейлистами
- Просмотр истории прослушиваний

### Для администратора
- Управление исполнителями (добавление, удаление)
- Управление альбомами (добавление, удаление)
- Управление треками (добавление с выбором MP3, удаление)
- Управление пользователями (просмотр, удаление)
- Управление тегами треков

## 🛠 Технологии

| Компонент             | Технология                             |
|-----------------------|----------------------------------------|
| Язык программирования | C# 10.0                                |
| Платформа             | .NET 6 / .NET 8                        |
| Интерфейс             | Windows Forms                          |
| База данных           | Microsoft SQL Server LocalDB           |
| Доступ к данным       | Entity Framework Core (Database First) |
| Хеширование паролей   | BCrypt.Net-Next                        |
| Воспроизведение аудио | NAudio                                 |
| Среда разработки      | Visual Studio 2022                     |

 

## 💻 Требования для запуска

- Операционная система: Windows 10 / 11 (64-bit)
- Процессор: от 1 ГГц
- Оперативная память: от 1 ГБ
- Звуковое устройство (колонки или наушники)
- Установленный SQL Server LocalDB
- .NET Runtime 6.0 или выше
